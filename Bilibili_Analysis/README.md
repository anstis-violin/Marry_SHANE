# B站热门视频数据分析项目

## 📊 项目简介

这是一个完整的B站热门视频数据分析项目，包含数据爬取、数据处理和可视化分析。

## 🎯 功能特点

- ✅ 使用B站官方API爬取排行榜数据
- ✅ 支持多个分区数据爬取
- ✅ 生成5个二维可视化图表
- ✅ 生成5个三维可视化图表
- ✅ 自动生成数据分析报告

## 📁 项目结构

```
Bilibili_Analysis/
├── crawler.py              # 数据爬虫脚本
├── visualize_2d.py         # 二维可视化脚本
├── visualize_3d.py         # 三维可视化脚本
├── run_all.bat             # 一键运行脚本
├── README.md               # 本文件
├── data/                   # 数据目录（自动创建）
│   ├── bilibili_data.csv   # CSV格式数据
│   ├── bilibili_data.json  # JSON格式数据

```

## 🚀 快速开始

### 方法1: 一键运行（推荐）

```bash

run_all.bat
```

### 方法2: 分步运行

```bash

# 2. 生成二维图表

# 3. 生成三维图表

```

## 📊 数据说明

### 爬取的数据字段

- **BV号**: 视频唯一标识
- **标题**: 视频标题
- **UP主**: 视频作者
- **播放量**: 视频播放次数
- **点赞数**: 点赞次数
- **投币数**: 投币次数
- **收藏数**: 收藏次数
- **分享数**: 分享次数
- **弹幕数**: 弹幕数量
- **分区**: 视频所属分区
- **点赞率/投币率/收藏率**: 互动率指标

### 支持的分区

- 全站排行榜
- 动画分区
- 音乐分区
- 游戏分区
- 娱乐分区

可以在 `crawler.py` 中修改 `categories` 列表来自定义爬取的分区。

## 📈 可视化图表说明

### 二维图表（5个）

1. **播放量分析** - 播放量分布、分区对比、Top 20视频、占比分析
2. **互动数据分析** - 点赞vs播放量、互动率对比、分区互动数据、相关性热力图
3. **UP主分析** - 高产UP主、热门UP主、平均播放量分布、视频数量分布
4. **分区雷达图** - 各分区多维度数据对比
5. **综合仪表盘** - 总体统计、分区对比、Top 10视频、互动率分布等

### 三维图表（5个）

6. **三维散点图** - 播放量-点赞数-投币数三维分布
7. **三维柱状图** - 各分区数据三维对比
8. **三维曲面图** - 播放量-点赞数-投币数关系曲面
9. **三维线框图** - 播放量-点赞数分布密度
10. **三维综合展示** - 多角度三维数据展示

## 💡 使用技巧

### 1. 修改爬取分区

在 `crawler.py` 中修改：

```python
categories = [
    {'rid': 0, 'name': '全站'},
    {'rid': 1, 'name': '动画'},
    # 添加更多分区...
]
```

### 2. 调整图表样式

在可视化脚本中可以修改：
- 颜色方案
- 图表大小
- 字体大小
- DPI分辨率

### 3. 数据分析

生成的CSV文件可以用Excel或其他工具进一步分析。

## 📝 注意事项

1. **遵守使用条款**: 使用B站官方API，遵守B站使用条款
2. **请求频率**: 代码中已设置1秒延迟，避免频繁请求
3. **数据时效性**: 排行榜数据实时更新，不同时间爬取结果可能不同
4. **网络连接**: 需要稳定的网络连接
