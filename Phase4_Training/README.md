# 美赛第四阶段训练 - 完整实操项目

## 📋 项目概述

这是一个完整的第四阶段训练项目，包含所有代码示例和实操任务。按照以下步骤完成训练。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 项目结构

```
Phase4_Training/

│   ├── task1_complex_3d_scene.py     # 任务1.1: 复杂三维场景
│   └── task2_interactive_3d.ipynb    # 任务1.2: 交互式三维图表

├── Day3_Data_Processing/             # Day 3: 数据获取和处理
│   ├── task1_web_scraper.py          # 任务3.1: 爬虫项目
```

## 📅 学习计划

### Day 1: 三维可视化（4-5小时）

**任务1.1: 复杂三维场景构建**
- 运行: `python Day1_3D_Visualization/task1_complex_3d_scene.py`
- 学习内容: 多对象三维场景、三维路径、等高线、体绘制

**任务1.2: 交互式三维可视化**
- 打开: `Day1_3D_Visualization/task2_interactive_3d.ipynb`
- 在VSCode中使用Jupyter扩展运行
- 学习内容: Plotly交互式三维图表

### Day 2: 高级二维可视化（4-5小时）

**任务2.1: 局部放大图高级应用**
- 运行: `python Day2_Advanced_2D/task1_zoom_inset.py`
- 学习内容: 多区域放大、嵌套放大

**任务2.2: 桑基图高级设计**
- 运行: `python Day2_Advanced_2D/task2_sankey_diagram.py`
- 学习内容: 多层级桑基图、交互式桑基图

**任务2.3: 其他高级二维图表**
- 运行: `python Day2_Advanced_2D/task3_other_charts.py`
- 学习内容: 雷达图、极坐标图

### Day 3: 数据获取和处理（4-5小时）

**任务3.1: 爬虫项目实战**
- 运行: `python Day3_Data_Processing/task1_web_scraper.py`
- 学习内容: 完整爬虫流程

**任务3.2: 数据处理和清洗**
- 运行: `python Day3_Data_Processing/task2_data_cleaning.py`
- 学习内容: 缺失值、异常值处理

### Day 4-5: 模型评估可视化（8-10小时）

**任务4.1: 混淆矩阵可视化**
- 运行: `python Day4_5_Model_Evaluation/task1_confusion_matrix.py`

**任务4.2: ROC曲线和AUC**
- 运行: `python Day4_5_Model_Evaluation/task2_roc_curve.py`

**任务4.3: 学习曲线**
- 运行: `python Day4_5_Model_Evaluation/task3_learning_curve.py`

**任务4.4: 综合可视化**
- 运行: `python Day4_5_Model_Evaluation/task4_comprehensive.py`
