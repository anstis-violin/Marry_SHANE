# 美赛第四阶段训练 - 完整使用指南

## 🎯 项目简介

这是一个完整的美赛代码手第四阶段训练项目，涵盖：
- ✅ 三维可视化（静态和交互式）
- ✅ 高级二维可视化（局部放大图、桑基图、雷达图等）
- ✅ 数据获取和处理（爬虫、数据清洗）
- ✅ 模型评估可视化（混淆矩阵、ROC曲线、学习曲线等）

## 🔧 常见问题

### Q1: 如何安装依赖包？
```bash
pip install -r requirements.txt
```

### Q2: 中文显示为方框怎么办？
代码中已经设置了中文字体（Microsoft YaHei），如果仍有问题，请检查系统是否安装了该字体。

### Q3: Jupyter Notebook如何使用？
在VSCode中安装Jupyter扩展，然后打开`.ipynb`文件即可运行。

### Q4: 爬虫无法访问网站怎么办？
代码中包含生成模拟数据的函数，取消注释即可使用。

### Q5: 图表保存失败怎么办？
确保对应的文件夹存在，或者修改保存路径。

### Q6: 如何修改图表样式？
修改代码中的颜色、字体、大小等参数即可。
